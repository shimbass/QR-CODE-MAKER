# QR 코드 제작기

React 기반의 친절한 UI를 가진 QR 코드 제작 프로그램입니다.

## 주요 기능

1. **이미지 중앙 배치**: 사용자가 업로드한 이미지를 QR 코드 중앙에 배치
2. **흰색 배경 처리**: 이미지 뒷 배경을 흰색으로 처리하여 QR 코드가 보이지 않도록 함
3. **링크 입력**: 복사 붙여넣기 가능한 링크 입력 필드
4. **드래그 앤 드롭**: 이미지를 드래그하여 쉽게 업로드 가능
5. **다중 생성**: 최대 10개까지 여러 QR 코드를 동시에 생성 가능
6. **일괄 저장**: 생성된 모든 QR 코드를 ZIP 파일로 한꺼번에 다운로드
7. **미리보기 모달**: 생성 완료 시 모든 QR 코드를 미리 확인 가능
8. **출력 기능**: 미리보기 모달에서 QR 코드를 인쇄 가능

## 설치 및 실행

### 1. 의존성 설치

```bash
npm install
```

### 2. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173` (또는 표시된 포트)로 접속하세요.

### 3. 프로덕션 빌드

```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

## 사용 방법

1. 링크 입력 필드에 URL을 입력하거나 붙여넣기
2. (선택) 중앙 이미지 업로드 - 클릭하거나 드래그 앤 드롭으로 이미지 선택
3. 필요시 [+추가] 버튼으로 최대 10개까지 링크 추가
4. [생성] 버튼 클릭
5. 자동으로 ZIP 파일이 다운로드되며, 미리보기 모달에서 생성된 QR 코드를 확인 가능
6. (선택) [출력] 버튼으로 QR 코드를 인쇄 가능

## 기술 스택

- **React 18**: UI 프레임워크
- **Vite**: 빠른 개발 환경 및 빌드 도구
- **Tailwind CSS**: 유틸리티 기반 CSS 프레임워크
- **qrcode**: QR 코드 생성 라이브러리
- **JSZip**: 여러 파일을 ZIP으로 압축
